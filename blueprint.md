
# 응모권 추첨기 애플리케이션

## 개요

이 애플리케이션은 사용자가 응모자 목록을 관리하고 목록에서 무작위로 우승자를 추첨할 수 있는 간단하고 인터랙티브한 웹 기반 응모권 추첨기입니다.

## 기능

-   **응모자 추가:** 사용자는 텍스트 입력 필드를 통해 응모자 목록에 새 이름을 추가할 수 있습니다.
-   **응모자 목록 표시:** 추가된 모든 응모자는 깔끔한 목록에 표시됩니다.
-   **우승자 추첨:** 버튼 클릭으로 목록에서 무작위 우승자를 선택합니다.
-   **우승자 표시:** 선택된 우승자는 축하 효과와 함께 눈에 띄게 표시됩니다.
-   **추첨 재설정:** 현재 응모자 목록과 우승자를 지우고 새로 시작합니다.
-   **더미 데이터:** 테스트를 위해 20개의 더미 이름을 목록에 추가하는 버튼을 제공합니다.

## 디자인 및 스타일

-   **모던 UI:** 깨끗하고 시각적으로 매력적인 레이아웃.
-   **인터랙티브 요소:** 버튼과 입력 필드에 호버 및 포커스 효과를 적용하여 사용자 경험을 향상시킵니다.
-   **색상 팔레트:** 생동감 있고 에너지 넘치는 색상 구성.
-   **타이포그래피:** 명확하고 읽기 쉬운 글꼴.
-   **애니메이션:** 우승자 발표 시 부드러운 애니메이션 효과를 추가합니다.
